The Game

Fermi Pico Bagel is a super fun number guess game for all ages. It is based on an old code-breaking game called Bulls and Cows.

How to play the game

A game leader writes a secret 4 digit number on paper. The game players take turns to guess what the secret number is. The game leader then responds each guess as follows:

  • If the guess has no matching numbers, the game leader responds "Bagel"
  • For each digit the guess has correct, but the position is wrong, the game leader responds "Pico"
  • For each digit the guess has correct, and in the correct position, the game leader responds "Fermi"

Example

As an example, let's say the secret number is 7685.

Guesses Respones Notes
1234 Bagel No digits is correct.
8697 Fermi Pico Pico 6 is in the right place, 7 and 8 are in the wrong places.
6598 Pico Pico Pico 5, 6 and 8 are all in wrong places.
7685 Fermi Fermi Fermi All digits are in the right place.

How to use the solver
  1. 1. Specify the number of digits (3 to 8), whether duplicate digits are allowed, zeros are allowed, and leading zero is allowed. Then click 'Start guesses'.
  2. 2. Start with a guess, and put in the response you get from the game leader. Use a "F" for each Fermi, a "P" for each Pico, and "B" for Bagel (for example "FPPB" is a valid response). Then click "Add Guess".
  3. 3. The remaining candidate numbers will be displayed in the "Candidates" box. The guess will also be added to the "Steps" table below, and a hint for the next guess will be displayed. Click on the hint to auto-fill the hint for the next turn.
  4. 4. Continue until there is only one candidate number left.
  5. 5. Click "New Game" for a new round.
